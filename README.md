# shorts-autoscroll
这是一个功能强大且高度可配置的 Auto.js 脚本，用于自动滑动短视频。脚本配备了一个悬浮窗控制面板。

## 🚀 功能特点

* **智能启动**：启动时自动拉起所选应用。
* **兼容双开**：独有的“应用监视器”功能，在您设置完毕后，它会自动等待（即使是双开应用选择弹窗），直到检测到目标应用已真正进入前台，才开始自动运行，全程无需手动点击“开始”。
* **悬浮窗控制**：
    * **实时日志**：在悬浮窗中显示详细的、自动滚动的操作日志（如：滑动次数、倒计时等）。
    * **状态显示**：固定显示总运行时长、屏幕分辨率、滑动坐标等信息。
    * **操作按钮**：提供“暂停/继续”和“停止脚本”的快捷按钮。
* **高度可配置（启动时）**：
    * **随机时间**：可自定义每个视频的最小和最大随机观看秒数。
    * **定时停止**：可设置脚本总运行分钟数。
    * **结束操作**：可自定义到时间后是“返回桌面并停止”还是“暂停播放并暂停脚本”。

## 🛠️ 运行环境

* **Auto.js (4.x.x 版本)**：本脚本在 Auto.js 4.x.x 免费版上开发和测试通过（由于使用了旧版 API，在高版本 Pro 上可能需要微调）。
* **Android 7.0+**
* **所需权限**：
    1.  **无障碍服务**：必须开启，用于执行 `swipe`（滑动）和 `home`（返回桌面）等操作。
    2.  **悬浮窗权限**：必须开启，用于显示控制面板和日志。

## ⚙️ 使用方法

1.  **准备环境**：
    * 安装 Auto.js 应用。
    * 在手机“设置”中，为 Auto.js 手动开启“无障碍服务”和“悬浮窗权限”。
2.  **运行脚本**：
    * 将本脚本（1.js）放入 Auto.js 的脚本目录中。
    * 在 Auto.js 应用中点击“运行”此脚本。
3.  **完成设置**：
    * 脚本会依次弹出对话框，请根据提示完成所有设置：
        1.  设置最小/最大观看秒数
        2.  设置总运行分钟数（0为不限制）
        3.  选择到时间后的结束操作
4.  **自动开始**：
    * 悬浮窗会显示“启动中...”。
    * 脚本会**自动检测**到应用已打开，并开始执行滑动。
5.  **控制**：
    * 在运行过程中，您可以随时通过悬浮窗上的按钮“暂停”或“停止”脚本。
    * 您可以按住悬浮窗顶部拖动它到任意位置。

## ⚠️ 注意事项

* 本脚本仅供个人学习和研究 Auto.js 自动化技术使用。
* 请遵守您所使用的短视频平台的用户协议。任何因滥用此脚本（如刷量、刷数据等）导致的账号限制或封禁，与本脚本作者无关。
